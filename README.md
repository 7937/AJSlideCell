#### How To Get Started

`JSlideGestureRecognizer *recognizer = [[JSlideGestureRecognizer alloc]initWithArray:textArr];`

`[cell addGestureRecognizer:recognizer];`

#### Installation

- Drag JSlideCell file to your project.
- `pod install 'JSlideCell'`
